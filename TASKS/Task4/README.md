## 🌦️ Weather Forecast Application (Python)

### 📌 Project Description

This is a **Python-based weather forecast application** that retrieves real-time weather information using the **OpenWeatherMap API**.
The application allows users to enter a city name or zip code and displays current weather details in a user-friendly format.

### 🚀 Features

* Fetches real-time weather data
* Displays temperature, humidity, wind speed, and weather conditions
* Accepts user input for city name or zip code
* Clear and readable output

### 🛠️ Technologies Used

* Python
* OpenWeatherMap API
* `requests` module
* `json` module

### ⚙️ How It Works

* Prompts the user to enter a city name or zip code
* Sends an API request using the `requests` library
* Parses the JSON response to extract relevant weather data
* Displays the retrieved weather information to the user

### 📂 Use Case

This project demonstrates API integration, JSON parsing, and handling real-world data using Python.

### ▶️ How to Run

1. Clone the repository
2. Install required library:

   pip install requests
   
3. Add your OpenWeatherMap API key in the script
4. Run the Python file:

   python weather_forecast.py
