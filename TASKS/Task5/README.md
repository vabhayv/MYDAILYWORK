🧠 Quiz Game (Python)

📌 Project Description

This is a Python-based quiz game that asks users a series of multiple-choice and fill-in-the-blank questions. The program is designed to make learning interactive by providing instant feedback and calculating the user’s final score at the end of the quiz.

Questions are stored in a list of dictionaries, making them easy to organize and manage. To make the game more engaging, the questions are randomly shuffled each time the program runs.

🚀 Features

Multiple-choice and fill-in-the-blank questions

Randomized question order for varied gameplay

Immediate feedback for correct and incorrect answers

Score tracking throughout the quiz

Final score calculation with performance message

Option to replay the quiz without restarting the program

🛠️ Technologies Used

Python

random module

⚙️ How It Works

Displays a welcome message and quiz instructions

Randomly shuffles the list of questions

Presents each question to the user one by one

Accepts user input and checks the answer

Provides instant feedback (Correct / Incorrect)

Calculates and displays the final score and performance summary

Prompts the user to play again if they wish

📂 File
quiz_game.py
▶️ How to Run

Clone the repository
